# Awesome M3U Editor

[![GitHub license](https://img.shields.io/github/license/arazgholami/awesome-m3u-editor)](https://github.com/arazgholami/awesome-m3u-editor/blob/main/LICENSE)
[![GitHub stars](https://img.shields.io/github/stars/arazgholami/awesome-m3u-editor)](https://github.com/arazgholami/awesome-m3u-editor/stargazers)

A powerful, web-based M3U playlist editor that allows you to manage and organize your IPTV playlists with ease. Edit, sort, and manage your M3U files directly in your browser with an intuitive interface.

🔗 **Live Demo**: [https://arazgholami.github.io/awesome-m3u-editor/](https://arazgholami.github.io/awesome-m3u-editor/)

## ✨ Features

- **Web-based** - No installation required, works directly in your browser
- **Intuitive UI** - Clean and responsive interface built with Bootstrap 5
- **Complete M3U Management**
  - Load and edit existing M3U/M3U8 files
  - Create and organize channel groups
  - Add, edit, and delete channels
  - Sort channels alphabetically
  - Move channels between groups
- **Live Preview** - Test channel URLs directly from the editor
- **Drag & Drop** - Reorder channels with simple drag and drop
- **Search & Filter** - Quickly find channels and groups
- **Offline Support** - Works completely client-side, no server required
- **Privacy Focused** - Your playlists never leave your computer

## 🚀 Getting Started

### Where to get playlists?
```
#All TV Channels grouped by category:
https://iptv-org.github.io/iptv/index.category.m3u

#All TV Channels grouped by language:
https://iptv-org.github.io/iptv/index.language.m3u

#All TV Channels grouped by country:
https://iptv-org.github.io/iptv/index.country.m3u
```

### Using the Web Version
1. Visit [https://arazgholami.github.io/awesome-m3u-editor/](https://arazgholami.github.io/awesome-m3u-editor/)
2. Click "Choose File" to upload your M3U file
3. Start editing your playlist
4. Click "Download Modified M3U" when done

### Running Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/arazgholami/awesome-m3u-editor.git
   cd awesome-m3u-editor
   ```
2. Open `index.html` in your web browser
3. Start editing your M3U files

## 🛠️ How to Use

1. **Upload Your M3U File**
   - Click "Choose File" and select your M3U/M3U8 file
   - The editor will parse and display your channels and groups

2. **Organize Channels**
   - Create new groups or use existing ones
   - Drag and drop channels to reorder them
   - Use the search box to filter channels

3. **Edit Channel Details**
   - Click on any channel to edit its details
   - Modify name, URL, and metadata
   - Use the preview button to test stream URLs

4. **Save Your Changes**
   - Click "Download Modified M3U" to save your changes
   - The file will be downloaded to your computer

## 🌟 Tips

- Use the search box to quickly find specific channels
- Double-click a group name to rename it
- Hold Shift while clicking to select multiple channels
- Use the keyboard arrows to navigate between channels
